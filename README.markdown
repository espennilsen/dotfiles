Initial setup:
==============
1. Clone it!

		git clone git://github.com/espennilsen/mydots.git ~/.mydots
		cd ~/.mydots
		git submodule init
		git submodule update

2. Link up the folders you want!

		ln -s ~/.mydots/.vim ~/.vim 
3. ?????
4. **Profit!**

Vim:
====
Remember to run :BundleInstall to tell Vundle to get the vim scripts specified in .vimrc
